Cara Memasang Ekstensi

  • Ekstrak file extensi.zip ke sebuah folder
  • Buka Chrome, ketik chrome://extensions di address bar lalu tekan Enter
  • Aktifkan Developer mode di pojok kanan atas
  • Klik Load unpacked, pilih folder hasil ekstrak yang berisi manifest.json
  • Klik ikon ekstensi di toolbar Chrome, tekan Jalankan Sekarang

*Saat pertama dibuka, klik "Pasang Ekstensi" di halaman selamat datang untuk melanjutkan.
